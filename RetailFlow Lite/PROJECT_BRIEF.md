# Tiny Python Data Engineering + Data Analysis Project

Project name: Mini Retail Sales Pipeline

Goal: Build a small Python-only data pipeline using loops and functions. You will read raw CSV files, clean the data, create a sales fact table, build simple KPI outputs, and save the results as CSV files.

Datasets:
- data/raw/customers.csv
- data/raw/products.csv
- data/raw/orders.csv

Recommended folders:
- data/raw/
- data/processed/
- reports/
- main.py

Focus:
- loops
- functions
- dictionaries/lists
- CSV reading/writing
- basic validation
- simple business KPIs

Do not use pandas for the first version. Use only Python standard libraries such as csv, datetime, collections, argparse, and pathlib.
