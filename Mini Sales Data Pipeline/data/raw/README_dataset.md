# Python Data Engineering + Data Analysis Mini Project Dataset

Files:
- raw_customers.csv: customer master data with one duplicate row and some missing cities.
- raw_products.csv: product master data with one missing category and one product where cost is higher than price.
- raw_orders.csv: order header data with duplicate order row, one invalid customer, and one invalid date.
- raw_order_items.csv: order line items with one invalid product, one duplicate row, and a few invalid quantities.

Business context: small online merchant sales data for January to March 2026.
Goal: use Python only to build a small CSV-based pipeline, clean data, create analytical tables, and generate KPIs.
